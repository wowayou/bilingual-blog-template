Before shaping a personal blog, it helps to study sites that have already been running for a while. The useful parts are often the quiet ones: archive density, topic indexes, search placement, post metadata, and the way a site stays readable over years.

## Primary references

- [Antfu - Posts](https://antfu.me/posts): a useful reference for archive density, year grouping, and lightweight page backgrounds.
- [Innei](https://innei.in/en): a useful reference for Light / System / Dark theme selection, article footer details, interaction entry points, and future extensibility.
- [Cassie Codes - Writing](https://www.cassie.codes/writing/): a useful reference for writing lists, dark mode, and long-page rhythm; this site will use load more for long listing pages.
- [Stitch via MCP](https://stitch.withgoogle.com/docs/mcp/setup): a reference for connecting design exploration with implementation workflows later.
