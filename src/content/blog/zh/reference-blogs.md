搭建个人博客之前，先看一些已经稳定运行的网站很有帮助。可以观察它们如何处理首页密度、文章归档、专题索引、搜索入口和个人介绍。

## 重点参考

- [Antfu - Posts](https://antfu.me/posts)：适合参考归档页的信息密度、年份分组和轻量背景处理。
- [Innei](https://innei.in/en)：适合参考 Light / System / Dark 主题选择、文章页底部信息、交互入口和未来可扩展能力。
- [Cassie Codes - Writing](https://www.cassie.codes/writing/)：适合参考写作列表、深色模式和长列表节奏，本站会用 load more 控制过长页面。
- [Stitch via MCP](https://stitch.withgoogle.com/docs/mcp/setup)：用于后续把设计发想、设计稿和代码实现衔接起来。
