This project entry tracks the site itself. The first pass focuses on making the core workflow solid rather than piling on features.

## Scope

- Astro static site
- Chinese `/zh/` and English `/en/` routes
- Markdown/MDX content collections
- Pagefind static search
- RSS, sitemap, canonical, and `hreflang`
- Giscus comment placeholder
- GitHub Actions plus Wrangler deployment
